# course-task-manager_36
**C语言实现的大学生课表&作业管理小助手**  
轻量离线工具，支持课表录入、作业追踪、到期提醒，适配Windows/macOS/Linux。


## 功能清单
 📅 课表管理：按周/日展示课程，支持添加/删除课程
 📝 作业记录：标记作业截止日期、优先级
 ⏰ 到期提醒：启动时弹窗提示即将到期的作业
 💾 本地存储：数据保存在`data.txt`，无需网络
